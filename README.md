# glassbox-ionic-cordova-new-app

## **Install dependecies:**

> Npm install

##  **Build app:**

**Android**

> ionic cordova build android

**iOS**

> ionic cordova build ios

## **Run app:**

**Android**

> ionic cordova run android --livereload -p=8080

**iOS**

> ionic cordova run ios --livereload -p=8080
